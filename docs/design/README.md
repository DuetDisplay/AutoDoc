# Design archive

Historical design specs and implementation plans written during AutoDoc's early
development (March 2026). These documents were created with AI-assisted planning
workflows and are preserved for context — they are **not** maintained as
up-to-date product documentation.

For current, accurate product behavior, see [`PRODUCT.md`](../../PRODUCT.md) in
the repository root.

## Contents

- [`plans/`](plans/) — step-by-step implementation plans with task checklists.
- [`specs/`](specs/) — design specifications for individual subsystems.

If you are self-hosting or deploying the calendar OAuth worker, follow
[`SELF_HOSTING.md`](../SELF_HOSTING.md) rather than the worker setup notes in
these archived specs.
